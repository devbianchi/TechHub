# TechHub

A modern technology solutions platform showcasing innovative products and services for the digital age.

<img src="/techhub.png" alt="Techhub preview">

## ✨ Features

- **Modern Tech Stack**: Built with React 18, Vite, and Tailwind CSS
- **Responsive Design**: Fully mobile-friendly with adaptive navigation
- **Smooth Animations**: Powered by Framer Motion for enhanced user experience
- **Dark Theme**: Professional dark mode with carefully crafted color palette
- **Category-based Navigation**: Filter products by AI, Cloud, Hardware, and Web solutions
- **Performance Optimized**: Code splitting with lazy loading for faster page loads

## 🚀 Pages

- **Home** - Hero section with features showcase and call-to-action
- **Products** - Browse and filter technology products by category
- **Services** - Explore our technology service offerings
- **About** - Learn about our mission, values, and statistics
- **Blog** - Technology insights and updates
- **Careers** - Join our team of technology experts
- **Contact** - Get in touch with our team

## 🛠️ Tech Stack

- **Framework**: React 18 + React Router 6
- **Build Tool**: Vite 5
- **Styling**: Tailwind CSS 3
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Backend Ready**: Express.js for future API integration

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/devbianchi/TechHub.git
cd techhub

# Install dependencies
npm install

# Start development server
npm run dev
```

## 🧑‍💻 Development

```bash
# Start dev server (http://localhost:5173)
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## 🎨 Color Palette

The project uses a custom color scheme with:

- **Accent**: Blue tones (#3b82f6)
- **Dark**: Grayscale with #0a0a0f as background
